Welcome to the ACCL Design Documentation Portal. This interactive viewer provides access to all architectural diagrams, sequence diagrams, wireframes, and database schemas created for the Adaptive Collaborative Code Learning Lab system. Navigate through each section to explore the comprehensive design artifacts that guide the implementation of this web-based programming education platform.

System Architecture Phase 3

The ACCL system follows a monolithic, layered architecture with MVC/MVT pattern organization. This diagram illustrates the interaction between the Flask web application, background workers, Docker sandboxes, and external service adapters.
